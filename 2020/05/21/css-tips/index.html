谷歌下font-size小于12px无效

这个bug只在PC端谷歌浏览器下才会出现，而且谷歌浏览器为中文版才有，英文版最小fontsize为10px

解决方案

  • 不去管，安卓和ios都会支持
  • transform: scale(.5);

transform的scale可以缩小放大容器，但是这个属性并不会改变原本标签（div之类）的大小，所以会出现留白
